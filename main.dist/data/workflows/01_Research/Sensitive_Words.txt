# ============================================
# HƯỚNG DẪN SỬ DỤNG
# ============================================
# - Từ khoa học (y học, sinh học) → GIỮ NGUYÊN (không thay thế)
# - Từ bạo lực/nhạy cảm → THAY THẾ theo TONE kịch bản
# - Format: TỪ_GỐC | HÀI_HƯỚC | NGHIÊM_TRỌNG | ĐỜI_THƯỜNG
# - (CẤM SỬ DỤNG) = Không dùng ở bất kỳ tone nào
# - (GIỮ NGUYÊN) = Thuật ngữ khoa học, OK dùng
# - (TRÁNH SỬ DỤNG) = Nên tránh hoàn toàn, có thể bị chặn

# 21. SAFE FRAMEWORK (ai-content-policy)
# ============================================
# Dùng khi viết prompt về chủ đề nhạy cảm (sức khỏe, y tế, cảnh báo)

# S - SPECIFY PURPOSE (Xác định mục đích)
#     → "For educational purposes", "For health awareness"
#     → Ví dụ: "Giáo dục sức khỏe về tác hại của đường"

# A - ADD CONTEXT (Thêm ngữ cảnh)
#     → Thêm background về nhu cầu
#     → Ví dụ: "Trong bối cảnh Tết, nhiều người ăn uống quá độ"

# F - FRAME APPROPRIATELY (Đặt khung phù hợp)
#     → Academic, professional, documentary
#     → Ví dụ: "Theo nghiên cứu y khoa, cơ chế như sau..."

# E - EXCLUDE HARMFUL (Loại bỏ chi tiết có hại)
#     → Không chi tiết đồ họa bạo lực
#     → Ví dụ: "Tập trung vào cơ chế, không mô tả đau đớn graphic"

# TEMPLATE ÁP DỤNG:
# "[S] Mục đích giáo dục sức khỏe,
#  [A] trong bối cảnh nghiên cứu về [topic],
#  [F] xin cung cấp phân tích khoa học về
#  [E] [topic] (tập trung cơ chế, không chi tiết bạo lực)."

# 23. ANTHROPOMORPHIC VIOLENCE SAFETY (🔴 CRITICAL)
# ============================================
# Mục đích: Tránh mô tả nhân vật nhân hóa (có mắt, miệng, cảm xúc)
# bị tra tấn, đau đớn, hoặc chịu bạo lực thể xác.
#
# 🔴 QUY TẮC: Khi nhân vật anthropomorphic (có eyes, mouth, arms) thì
# nhân vật đó = SINH VẬT SỐNG. Mọi hành vi gây đau đớn lên nhân vật đó
# sẽ bị AI platforms (Gemini, Veo, YouTube) coi là MÔ TẢ TRA TẤN.
#
# ❌ CẤM TUYỆT ĐỐI (Nhân vật nhân hóa KHÔNG ĐƯỢC):
# - Bị bỏ vào lò/microwave/nước sôi (= tra tấn bằng nhiệt)
# - Hiển thị pained grimace, tears, flailing helplessly (= đau đớn)
# - Melting, dissolving, crumbling (= biến dạng cơ thể do bạo lực)
# - Bị cắt, chặt, thái, xay (= bạo lực thể xác)
# - Bị đốt, cháy, thiêu (= tra tấn bằng lửa)
# - Bị nghiền, ép, dập (= tra tấn bằng áp lực)
#
# ✅ CÁCH LÀM ĐÚNG:
# - Nhân vật ĐỨNG BÊN CẠNH và QUAN SÁT hành vi sai
# - Nhân vật BÌNH LUẬN/PHÀN NÀN (annoyed, disapproving, unimpressed)
# - Nhân vật DẠY CÁCH LÀM ĐÚNG (teaching, instructing)
# - Dùng expression AN TOÀN: frown, sigh, arms crossed, head shake
#
# ❌ FORBIDDEN VISUAL EXPRESSIONS (cho nhân vật nhân hóa):
# pained grimace | tears forming | flailing helplessly
# melting unevenly | drooping | writhing | screaming
# burning | dissolving | crumbling | being cooked
#
# ⚠️ LƯU Ý: Lời thoại (dialogue) KHÔNG bị hạn chế bởi rule này.
# Chỉ VISUAL DESCRIPTIONS trong Master/Prompts mới cần kiểm tra.
#
# ✅ SAFE VISUAL ALTERNATIVES:
# pained grimace    → annoyed disapproving frown
# tears forming     → cartoon sigh bubble above head
# flailing helplessly → arms crossed shaking head
# melting unevenly  → (REMOVE - nhân vật không ở trong lò)
# drooping          → arms on hips unimpressed
#
# ⚠️ ĐẶC BIỆT CHÚ Ý VỚI CÁC CHỦ ĐỀ:
# - Đông lạnh/rã đông (thịt, cá) → Nhân vật quan sát, không bị rã đông
# - Chiên/rán/nướng → Nhân vật dạy cách, không bị chiên
# - Luộc/hấp → Nhân vật hướng dẫn, không bị luộc
# - Cắt thái → Nhân vật giải thích, không bị cắt
